## 🛠 SYSTEM OVERVIEW
Z.AI Emergency Portal is an agentic AI-driven ecosystem designed for real-time flood rescue coordination. It addresses critical communication blackouts during disasters like the 2021 Sri Muda floods.

### Key Features:
* **Agentic Priority Engine:** Automatically categorizes SOS signals (P0-P3) using spatial reasoning (e.g., "Knee-deep on Roof" = P0).
* **Dead Phone Protocol:** Low-power mode activation and visual beacon pulsing for last-mile rescue.
* **Offline Resilience:** Simulated mesh network protocol for store-and-forward data transmission during network failure.
* **Real-time Synchronization:** Powered by Firebase for zero-latency command and control updates.
